# netfunsmooth

R package for network-weighted smoothing of functional data.

## Project

This repository contains the implementation developed as part of my bachelor thesis at LMU Munich.

The goal is to investigate how functional observations assigned to nodes of a graph can be smoothed by using temporal smoothness and neighborhood information from the network structure.

## Project Structure

- `R/`: package functions
- `scripts/`: simulation and analysis scripts
- `tests/`: unit tests
- `vignettes/`: reproducible examples
- `data/`: example data

This repository is under active development.
